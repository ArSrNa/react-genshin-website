# 部署与安装

## 技术栈

项目由next与Typescript编写

## 安装

进入项目目录，`npm i`安装所需依赖


使用`npm run dev`启动next项目，并监听`3000`端口，可在浏览器打开`http://localhost:3000`查看效果。
